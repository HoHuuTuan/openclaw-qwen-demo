OpenClaw + Qwen 120k Context Optimization Demo

Demo này chứng minh rằng Qwen 120k context window không đồng nghĩa usable context trong OpenClaw agent runtime cũng là 120k. Bản demo mặc định chạy bằng mock mode, không cần key thật.

Why Qwen 120k can still crash in OpenClaw

Vấn đề không nằm ở model window trên giấy, mà nằm ở lượng context bị inject mỗi turn: SOUL.md, AGENTS.md, MEMORY.md, TOOLS.md, history, tool outputs, logs và search results.

How openclaw-optimization-guide solves it

Giải pháp là giữ prompt files cực nhỏ, chuyển knowledge dài sang vault, ép sub-agent gom output lớn thành summary, rồi pruning và compaction trước khi gửi tới model backend.
